# Price Aggregator v5 — веб-интерфейс для команды

Это не «скрипт для программиста», а простой веб-интерфейс:
- открывается в браузере
- есть дашборд
- есть страницы по шагам
- сотрудники работают через кнопки и формы

## Как запускать
### Локально
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Как это выглядит для команды
После запуска открывается веб-страница.
Дальше команда работает только в браузере:
- Загрузка прайсов
- Обработка
- Дубли
- Итоговый прайс

## Что дальше можно сделать
Эту версию легко вынести на сервер и дать доступ по ссылке всей команде.
