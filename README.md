# Price Aggregator

Локальный агрегатор прайсов для 4 поставщиков.

## Возможности
- загрузка файла вручную
- загрузка по ссылке
- нормализация разных прайсов
- заготовка под fuzzy matching
- SQLite база
- Streamlit интерфейс

## Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
```
